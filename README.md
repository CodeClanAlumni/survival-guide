## Set-up

* Check you have ruby version 2.x.x or above
```
$ ruby --version
```

* Install the Bundler
```
$ gem install bundler
```

* Install the plugins ([Jekyll](https://jekyllrb.com/) and its dependencies)
```
$ bundle install
```

* Run locally
```
$ bundle exec jekyll serve
```

* Head to `localhost:4000` or `http://127.0.0.1:4000`
